🚀 خطوات سريعة لتفعيل Google Sign-In على Vercel

═══════════════════════════════════════════════════════════

📍 الخطوة 1: اعرف دومين Vercel الخاص بك
───────────────────────────────────────────────────────────
1. اذهب إلى: https://vercel.com/dashboard
2. اختر مشروعك
3. انسخ الدومين (مثلاً: my-platform.vercel.app)

═══════════════════════════════════════════════════════════

🔥 الخطوة 2: Firebase Console
───────────────────────────────────────────────────────────
1. افتح: https://console.firebase.google.com/
2. اختر: a-platform-for-learning
3. Authentication > Settings > Authorized domains
4. Add domain
5. أضف: your-domain.vercel.app
6. Add

═══════════════════════════════════════════════════════════

☁️ الخطوة 3: Google Cloud Console
───────────────────────────────────────────────────────────
1. افتح: https://console.cloud.google.com/
2. APIs & Services > Credentials
3. اضغط على OAuth 2.0 Client ID

في Authorized JavaScript origins:
   + ADD URI: https://your-domain.vercel.app

في Authorized redirect URIs:
   + ADD URI: https://your-domain.vercel.app/__/auth/handler

4. SAVE

⚠️ مهم: لا تنسى /__/auth/handler في النهاية!

═══════════════════════════════════════════════════════════

⏱️ الخطوة 4: انتظر
───────────────────────────────────────────────────────────
انتظر 5-10 دقائق حتى تنتشر التغييرات

═══════════════════════════════════════════════════════════

🧹 الخطوة 5: مسح الـ Cache
───────────────────────────────────────────────────────────
Ctrl + Shift + Delete
أو جرب في Incognito Mode

═══════════════════════════════════════════════════════════

🧪 الخطوة 6: اختبار
───────────────────────────────────────────────────────────
1. افتح: https://your-domain.vercel.app
2. اذهب لصفحة تسجيل الدخول
3. اضغط "تسجيل الدخول بـ Google"
4. يجب أن يعمل! ✅

═══════════════════════════════════════════════════════════

📝 مثال:
───────────────────────────────────────────────────────────
إذا كان دومين Vercel: learning-platform.vercel.app

Firebase Console:
  ✅ learning-platform.vercel.app

Google Cloud Console:
  JavaScript origins:
    ✅ https://learning-platform.vercel.app
  
  Redirect URIs:
    ✅ https://learning-platform.vercel.app/__/auth/handler

═══════════════════════════════════════════════════════════

🐛 الأخطاء الشائعة:
───────────────────────────────────────────────────────────
❌ redirect_uri_mismatch
   → تأكد من إضافة /__/auth/handler

❌ unauthorized-domain
   → تأكد من إضافة الدومين في Firebase

❌ This app is blocked
   → غيّر OAuth consent screen إلى Production

✓ كل شيء يعمل بشكل مثالي!
